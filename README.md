# Agent vs Workflow Explainer

A WebGPU-based agent explainer. Gemma runs entirely in your browser — no backend — to visualize how LLM agents and workflows run, step by step.

Purpose of this repo is for education purpose, the small harness existing in the repo is only purposed to simulate agent loop for demostration purpose.
