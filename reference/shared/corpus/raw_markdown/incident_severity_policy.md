# Incident Severity Policy

- **Sev-1:** full outage or data loss; page immediately, all hands.
- **Sev-2:** major degradation; page on-call.
- **Sev-3:** minor degradation; handle in business hours.
- **Sev-4:** cosmetic; backlog it.
Set severity by user impact, not by how loud the alert is.
