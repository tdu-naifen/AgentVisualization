# On-Call Runbook Index

- High CPU -> CPU Saturation Root-Cause Runbook
- High memory -> Memory Leak / OOM Runbook
- p99 latency -> Network Latency / GC Pause / DB Pool runbooks
- Shard restart error E1342 -> Shard Restart Error E1342 Runbook
- Disk full -> Disk Full Runbook
- 429 storm -> Upstream Rate-Limit Runbook
