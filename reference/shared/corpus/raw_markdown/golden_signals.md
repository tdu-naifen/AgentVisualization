# The Four Golden Signals

1. **Latency:** how long requests take (watch the tail, not the mean).
2. **Traffic:** demand on the system.
3. **Errors:** rate of failed requests.
4. **Saturation:** how full the most constrained resource is (often CPU or I/O).

Start every investigation by checking these four.
