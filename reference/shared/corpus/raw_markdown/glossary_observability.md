# Glossary: Observability Terms

- **SLI/SLO/SLA:** indicator, objective, agreement.
- **p99:** the 99th-percentile value; the tail users feel.
- **Cardinality:** number of distinct tag combinations; high cardinality is costly.
- **Saturation:** how full a resource is; one of the four golden signals.
- **Trace:** the path of one request across services.
